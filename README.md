# Ionic-App "Tierheimtiere" #

<br>

This repository contains an [Ionic app](https://ionicframework.com/) with [Angular](https://angular.io/) that demonstrates how
[ion-tabs](https://ionicframework.com/docs/api/tabs) can be used.
